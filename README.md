# 🚗 AutoLuxDetail — Premium Car Detailing Website

A high-end, modern web experience for a **premium eco-friendly car detailing service**.
Built with performance, motion design, and conversion-focused UX in mind.

---

## ✨ Overview

AutoLuxDetail is a **luxury-grade frontend application** designed to showcase and sell car detailing services with:

* Cinematic UI/UX
* Smooth animations (Framer Motion)
* Conversion-driven components
* Mobile-first responsive design

---

## 🚀 Features

### 🎯 Core Sections

* Hero Section (parallax + interactive lighting)
* Services & Process Flow
* Pricing System (filterable + CTA-driven)
* Gallery (lightbox + immersive view)
* Testimonials (animated + marquee)
* FAQ (searchable + interactive)
* Contact + Footer (lead capture + CTA)

---

### ⚡ Advanced UX Features

* Scroll-based animations
* Dynamic navbar with active tracking
* Smooth anchor navigation
* Glassmorphism + gradient UI system
* Fully responsive (mobile-first)
* Micro-interactions across all components

---

## 🛠 Tech Stack

| Category  | Technology                 |
| --------- | -------------------------- |
| Framework | React (Vite / Next-ready)  |
| Styling   | Tailwind CSS               |
| Animation | Framer Motion              |
| Icons     | Lucide React               |
| Routing   | React Router               |
| Utilities | Custom `cn()` class merger |

## ⚙️ Installation & Setup

```bash
# Clone the repo
git clone https://github.com/your-username/autolux-detail.git

# Navigate
cd autolux-detail

# Install dependencies
npm install

# Run dev server
npm run dev
```
